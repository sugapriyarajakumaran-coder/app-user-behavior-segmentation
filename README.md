# 📱 App User Behavior Segmentation

A Machine Learning project that segments mobile app users into different behavioral groups using Unsupervised Learning techniques.

---

# 🚀 Project Overview

This project analyzes user activity data and groups users based on their engagement patterns, session behavior, activity levels, and churn indicators.

The goal is to help businesses:

- Identify high-value users
- Detect churn-risk users
- Improve retention strategies
- Create personalized marketing campaigns
- Understand customer behavior patterns

---

# 🧠 Machine Learning Techniques Used

## ✅ K-Means Clustering
Used to segment users into different engagement groups.

## ✅ PCA (Principal Component Analysis)
Used for dimensionality reduction and cluster visualization.

## ✅ MiniBatchKMeans
Used for scalable clustering on large datasets.

## ✅ Gaussian Mixture Model (GMM)
Used for probabilistic clustering comparison.

---

# 📊 User Segments Identified

| Cluster | User Type | Description |
|---|---|---|
| Cluster 0 | High Users | Highly engaged users with frequent app activity |
| Cluster 1 | Moderate Users | Users with balanced and consistent activity |
| Cluster 2 | Low Users | Less active users with lower engagement |
| Cluster 3 | Occasional Users | Users with irregular or infrequent usage |

- Crash events
- Support tickets
- Engagement score
- Churn risk score
- Account age
